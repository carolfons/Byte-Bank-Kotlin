  <h2>:bank:Descrição do projeto </h2>
  <p> Projeto criado tem como objetivo simular um banco online, onde se é possível ter acesso a sua conta-corrente digital.
  O projeto possui os atributos:
  <ul>
    <li>Titular</li>
    <li>Conta</li>
   <li>Saldo</li>
    </ul>
  E os seguintes métodos:
  <ul>
    <li>Depositar</li>
   <li>Sacar</li>
   <li>Transferir</li>
 </ul>
